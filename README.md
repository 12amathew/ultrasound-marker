# Ultrasound Marker

Ultrasound Marker is a desktop app for preparing, marking, resolving, and exporting ultrasound practical assessment results.

The detailed documentation is in [`docs/`](./docs/README.md):

- [`docs/admin-user-guide.md`](./docs/admin-user-guide.md): plain-language guide for assessment admins.
- [`docs/technical-architecture.md`](./docs/technical-architecture.md): architecture and maintenance guide for technical readers.

## Development

Install dependencies:

```bash
npm install
```

Run the app in development:

```bash
npm run dev
```

Build the app:

```bash
npm run build
```
